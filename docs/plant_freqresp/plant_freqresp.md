# 1軸テーブルのプラント特性の解釈

水平方向に動く1軸のテーブルについて、基本的なバネ・マス・ダンパ系でモデルした際のプラントの周波数特性を解説。

## 教科書的なおさらい

標準的なバネマスダンパ系の式から周波数伝達関数の導出まで。

不要な人は[設定の導入](#設定の導入)まで

### 運動方程式

$$ m\ddot{x}(t) + c\dot{x}(t) + kx(t) = f(t) $$

### 伝達関数の導出

初期状態を静止状態（$x(0)=0, \dot{x}(0)=0$）と仮定してラプラス変換する。

$X(s), F(s)$はそれぞれ変位$x(t)$と入力$f(t)$のラプラス変換である。

```math
\begin{align}
ms^2X(s) + csX(s) + kX(s) = F(s)\\
(ms^2+cs+k)X(s) = F(s)
\end{align}
```

伝達関数$G(s)$は入力に対する出力の比なので、

$$ G(s) = \frac{X(s)}{F(s)}$$

よって、伝達関数は以下の通り。

$$ G(s) = \frac{1}{ms^2+cs+k}$$

### 周波数伝達関数への変換

このプラントモデルのBode線図を描く。

周波数伝達関数に変換するため、ラプラス演算子を$ j\omega $に置き換える。

```math
s → j\omega
```

このとき、伝達関数は以下の通り変換される。

```math
G(j\omega) = \frac{1}{m(j\omega)^2 + c(j\omega) + k} = \frac{1}{(k - m\omega^2) + j(c\omega)}
```

#### ゲイン$G_{dB}$の計算

Bode線図のゲイン特性は入出力の振幅比を`dB`で表したもの。

```math
|G(j\omega)| = \frac{1}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}}
```

```math
G_{dB} = 20 \log_{10} |G(j\omega)|
```

この式から、以下のことが分かる。

- 低周波：$20 \log_{10}(1/k)$に収束。
  - $\omega \to 0$のとき
- 共振点：分母が最小となり、グラフにピークが現れる。
  - $\omega = \sqrt{k/m}$のとき
- 高周波：$-40\,dB/dec$ の傾きで減衰。
  - $\omega \to \infty$のとき

#### 位相$\phi$の計算

Bode線図の位相特性は入力に対して出力がどれだけ遅れるかを度数[°, deg]で表したもの。

$G(j\omega)$を複素平面上のベクトルとして考え、その偏角を求める。

```math
\phi = \angle G(j\omega) = -\arctan \left( \frac{c\omega}{k - m\omega^2} \right)
```

この式から、以下のことが分かる。

- 低周波: 位相遅れは $0^\circ$。
- 共振点: 位相がちょうど $-90^\circ$ 遅れる。
- 高周波: 最大で $-180^\circ$ まで遅れる（2次遅れ系）。

## 設定の導入

ここから制御対象の設定を深めていきます。

- 水平方向に動く1軸テーブルをサーボモーターで制御する。
- モーターはダイレクトドライブのリニアサーボモータを使用する。
- ガイドはリニアガイドを使用する。
- 可動部にリニアスケールを貼り付け、ベース側に取り付けられたリードヘッドで位置を読み取る。
- 
![Modeling Plant](plant_modeling.drawio.svg)

- m：可動部質量 [kg]
- c：リニアガイドの粘性摩擦係数
- k：ばね定数
- f：サーボ推力, 入力
- x：テーブル位置, 出力

本件では十分な剛性があるものとして、以降`k = 0`とします。

### 伝達関数

```math
G(s) = \frac{1}{ms^2 + cs} = \frac{1}{s}\cdot\frac{1}{(ms + c)}
```

`k = 0`のときは積分要素$1/s$と1次遅れ要素（$1/(ms+c)$が直列に並んだ形になる。

### 周波数伝達関数

#### ゲイン特性

```math
|G(j\omega)| = \frac{1}{\sqrt{(-m\omega^2)^2 + (c\omega)^2}} = \frac{1}{\omega \sqrt{m^2\omega^2 + c^2}}
```

低域($\omega \to 0$)では、慣性項 $m\omega$ が粘性抵抗 $c$ に対して無視できるほど小さい状態なので、以下のように近似する。

```math
\sqrt{m^2\omega^2 + c^2} \approx \sqrt{c^2} = c
```

これを元の式に代入すると：

```math
|G(j\omega)| \approx \frac{1}{c\omega}
```

```math
G_{dB} = -20 \log_{10}(\omega) - 20 \log_{10}(c)
```

であるから、Bode線図は`-20dB/dec`の傾きを示す。

    シンプルに「$\omega$に反比例である」って言ったりする。

高域($\omega \to \infty$)では、慣性項 $m\omega$ が粘性抵抗 $c$ よりも支配的になるため、以下のように近似できる。

```math
\sqrt{m^2\omega^2 + c^2} \approx \sqrt{m^2\omega^2} = m\omega
```

これを元の式に代入すると：

```math
|G(j\omega)| \approx \frac{1}{\omega \cdot m\omega} = \frac{1}{m\omega^2}
```

```math
G_{dB} = -40 \log_{10}(\omega) - 20 \log_{10}(m)
```

であるから、Bode線図は`-40dB/dec`の傾きを示す。

    こっちは$\omega^2$に反比例であるってあまり言わないかも…
    個人的には-40dB/decって言うことが多い

#### 位相特性

```math
\angle G(j\omega) = -90^\circ - \arctan\left(\frac{m\omega}{c}\right)
```

式から、-90°の積分要素と周波数に応じて変化する一時遅れから構成されることがわかる。

Bode線図上では、低周波側で-90°に漸近し、高周波側は-180°に漸近する。

## プロットの確認

可動部を横100mm, 奥行き100mm, 高さ50mmのアルミブロックとし、`m = 3.5`に設定。

機械剛性は十分高いものとして`k = 0`に設定。

リニアガイドの抵抗は`c = 10`に設定。

設定はあくまで一例。

```MATLAB
% Parameter
m = 3.5; % = 100[mm] * 100[mm] * 50[mm] * 2.7*10^3[kg/m^3]
c = 10;
k = 0;

% TF
num = 1.0;
den = [m c k];
sys_plant = tf(num, den);

% Bodeplot
bodeplot(sys_plant);
```

![MATLAB Plot: Plant FR](plant_frd.png)

周波数が低域から高域に移るに連れて、ゲインは傾きが-20dB/decから-40dB/decに変化していることが分かる。

同様に位相は-90°から-180°に変化している。
